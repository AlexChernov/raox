Анимация

Описание кадра анимации имеет следующий вид:

frame имя_кадра {
    тело_кадра
}

В процессе прогона в окне анимации отображается выбранный на данный момент кадр из списка всех кадров, описанных для модели. В теле каждого кадра с помощью процедурного языка и специальных функций для отрисовки объектов описывается изображение кадра.
Определены следующие функции для отрисовки объектов:

Сигнатура Описание Параметры 
color(
        int r,
        int g,
        int b
        [, int alpha])
 Задание цвета в цветовой модели RGB. Используется для описания цвета в остальных функциях отрисовки.
r интенсивность красного цвета в интервале от 0 до 255
g интенсивность зеленого цвета в интервале от 0 до 255
b интенсивность синего цвета от в интервале 0 до 255
alpha задание прозрачности в интервале от 0 до 255 
background(
        int width,
        int height,
        color c)
 Задание фона кадра. Функция задания фона должна вызываться до всех остальных функций отрисовки
width ширина кадра
height высота кадра
c цвет кадра 
text(
        double x,
        double y,
        double width,
        double height,
        color fillColor,
        color textColor,
        String text
        [, alignment a])
 Отрисовка текста
x координата x верхнего левого угла панели текста
y координата y верхнего левого угла панели текста
width ширина панели текста
width высота панели текста
fillColor цвет фона панели текста
textColor цвет текста
text текст
alignment выравнивание текста в панели, задается одним из значений left, center или right 
line(
        double x1,
        double y1,
        double x2,
        double y2,
        color c)
 Отрисовка отрезка
x1 координата x первой точки отрезка
y1 координата y первой точки отрезка
x2 координата x второй точки отрезка
y2 координата y второй точки отрезка
c цвет отрезка 
rectangle(
        double x,
        double y,
        double width,
        double height,
        color fillColor,
        color borderColor)
 Отрисовка прямоугольника
x координата x левой верхней точки прямоугольника
y координата y левой верхней точки прямоугольника
width ширина прямоугольника
height высота прямоугольника
fillColor цвет фона прямоугольника
borderColor цвет границы прямоугольника 
circle(
        double x,
        double y,
        double radius,
        color fillColor,
        color borderColor)
 Отрисовка круга
x координата x центра круга
y координата y центра круга
radius радиус круга
fillColor цвет фона круга
borderColor цвет границы круга 
ellipse(
        double x,
        double y,
        double width,
        double height,
        color fillColor,
        color borderColor)
 Отрисовка эллипса
x координата x левого верхнего угла прямоугольника, в который вписан эллипс
y координата y левого верхнего угла прямоугольника, в который вписан эллипс
width длина горизонтальной оси эллипса
height длина вертикальной оси эллипса
fillColor цвет фона эллипса
borderColor цвет границы эллипса 
triangle(
        double x1,
        double y1,
        double x2,
        double y2,
        double x3,
        double y3,
        color fillColor,
        color borderColor)
 Отрисовка треугольника
x1 координата x первой вершины треугольника
y1 координата y первой вершины треугольника
x2 координата x второй вершины треугольника
y2 координата y второй вершины треугольника
x3 координата x третьей вершины треугольника
y3 координата y третьей вершины треугольника
fillColor цвет фона треугольника
borderColor цвет границы треугольника

Пример:

event timer() {
    timer.plan(currentTime + 1);
}

set init() {
    timer.plan(currentTime + 1);
}

set terminateCondition() {
    return currentTime > 200;
}

frame whosthat {
    background(300, 300, color(255, 255, 255));

    double dy = Math.abs(4 - currentTime % 8);
    double dx = Math.abs(8 - currentTime % 16);

    /* tail */
    triangle(255, 170 + dy, 290, 40 + dy, 200, 150 + dy, color(255, 153, 188), color(255, 153, 188));
    line(255, 170 + dy, 290, 40 + dy, color(0, 0, 0));

    triangle(100, 20 + dy, 290, 40 + dy, 275, 75 + dy, color(255, 153, 188), color(255, 153, 188));
    triangle(100, 20 + dy, 140, 24 + dy, 130, 30 + dy, color(255, 255, 255), color(255, 255, 255));
    line(100, 20 + dy, 290, 40 + dy, color(0, 0, 0));
    line(100, 20 + dy, 265, 72 + dy, color(0, 0, 0));

    triangle(250, 220 + dy, 270, 40 + dy, 200, 150 + dy, color(255, 153, 188), color(255, 153, 188));
    line(250, 68 + dy, 220, 120 + dy, color(0, 0, 0));

    /* right back leg */
    triangle(220, 220 + dy, 245 + dx, 255 + dy, 260, 183 + dy, color(255, 153, 188), color(255, 153, 188));

    /* body */
    ellipse(30, 100 + dy, 230, 150, color(255, 153, 188), color(0, 0, 0));

    /* right back leg partially-behind effect */
    triangle(240, 220 + dy, 245 + dx, 255 + dy, 260, 183 + dy, color(255, 153, 188), color(255, 153, 188));
    triangle(233, 243 + dy, 245 + dx, 255 + dy, 255, 245 + dy, color(255, 255, 255), color(255, 255, 255));
    line(226, 228 + dy, 245 + dx, 255 + dy, color(0, 0, 0));
    line(245 + dx, 255 + dy, 260, 183 + dy, color(0, 0, 0));

    /* left ear */
    circle(35, 150 + dy, 40, color(255, 153, 188), color(0, 0, 0));

    /* head */
    ellipse(7, 127 + dy, 135, 125, color(255, 153, 188), color(0, 0, 0));

    /* right ear */
    circle(120, 150 + dy, 40, color(255, 153, 188), color(0, 0, 0));
    circle(120, 150 + dy, 20, color(255, 153, 188), color(0, 0, 0));

    /* left eye */
    circle(23, 180 + dy, 30, color(255, 255, 255), color(0, 0, 0));
    circle(28, 180 + dy, 2, color(0, 0, 0), color(0, 0, 0));

    /* right eye */
    circle(80, 180 + dy, 30, color(255, 255, 255), color(0, 0, 0));
    circle(78, 180 + dy, 2, color(0, 0, 0), color(0, 0, 0));

    /* left front leg */
    triangle(10, 225 + dy, 1 - dx, 265 + dy, 80, 225 + dy, color(255, 153, 188), color(255, 153, 188));
    triangle(0, 255 + dy, 1 - dx, 265 + dy, 15, 258 + dy, color(255, 255, 255), color(255, 255, 255));
    line(10, 225 + dy, 1 - dx, 265 + dy, color(0, 0, 0));
    line(1 - dx, 265 + dy, 80, 225 + dy, color(0, 0, 0));

    /* mouth area */
    ellipse(1, 188 + dy, 100, 70, color(242, 245, 207), color(0, 0, 0));
    triangle(10, 225 + dy, 45, 250 + dy, 70, 225 + dy, color(255, 153, 188), color(0, 0, 0));

    /* nose */
    line(20, 214 + dy, 27, 217 + dy, color(0, 0, 0));
    line(38, 217 + dy, 44, 214 + dy, color(0, 0, 0));

    /* teeth */
    triangle(15, 225 + dy, 20, 230 + dy, 25, 225 + dy, color(255, 255, 255), color(0, 0, 0));
    triangle(50, 225 + dy, 55, 230 + dy, 60, 225 + dy, color(255, 255, 255), color(0, 0, 0));

    /* right front leg */
    triangle(135, 240 + dy, 170 + dx, 275 + dy, 175, 225 + dy, color(255, 153, 188), color(255, 153, 188));
    triangle(159, 265 + dy, 170 + dx, 275 + dy, 176, 260 + dy, color(255, 255, 255), color(255, 255, 255));
    line(135, 240 + dy, 170 + dx, 275 + dy, color(0, 0, 0));
    line(175, 225 + dy, 170 + dx, 275 + dy, color(0, 0, 0));
}
