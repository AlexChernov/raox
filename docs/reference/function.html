    • Функции

    Описание функции имеет следующий вид:

    тип_возвращаемого_значения имя_функции(описание_параметров_функции) {
    тело_функции
}

    Вызов функции имеет следующий вид:

    имя_функции(список_значений_параметров_функции);

    Функции могут вызываться из любых элементов модели.
    Пример вызова одной функции из другой:

    double длина_линии(double x1, double y1, double x2, double y2) {
    double y_diff = Math.abs(y2 - y2);
    double x_diff = Math.abs(x2 - x1);
    return Math.sqrt(y_diff * y_diff + x_diff * x_diff);
}

double периметр_треугольника(
        double x1, double y1,
        double x2, double y2,
        double x3, double y3
) {
    return длина_линии(x1, y1, x2, y2) + длина_линии(x2, y2, x3, y3)
            + длина_линии(x1, y1, x3, y3);
}

    Пример вызова функции для подсчета результата:

    int определить_победителя() {
    if (игрок_1.сумма_очков > игрок_2.сумма_очков)
        return 1;
    else if (игрок_1.сумма_очков < игрок_2.сумма_очков)
        return 2;
    else
        return 0;
}

result победитель = getValue(определить_победителя());
