    • Константы

    Объявление глобальных констант модели осуществляется с помощью ключевого слова constant. Значение константы должно быть задано при ее объявлении.

    constant тип_значения_константы имя_константы = значение_константы;

    Примеры:

    constant double pi = 3.14159;
constant double e = 2.71828;
    enum этап_обработки {накопитель_1, станок_1, манипулятор_1, станок_2, накопитель_2};

constant этап_обработки начальное_положение_заготовки = этап_обработки.накопитель_1;
