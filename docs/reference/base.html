    Встроенные типы данных

    Тип данных Описание 
    int
    		 Целочисленный тип данных. 
    double
    		 Числа с плавающей точкой. 
    boolean
    		 Логический тип данных. 
    String
    		 Строковый тип данных. 
    enum
    		 Перечислимые типы данных. Разработчик имеет возможность объявлять новые перечислимые типы. Использование значения перечислимого типа осуществляется с помощью конструкции имя_перечислимого_типа.значение_перечислимого_типа.

    Пример:

    enum Тип_заготовки {вал, диск};
enum Тип_станка {фрезерный, токарный};

type Заготовка {
    int номер;
    double вес;
    boolean срочная;
    Тип_заготовки тип;
}

type Станок {
    int номер;
    String название;
    Тип_станка тип;
}

Заготовка.create(1, 0.420, false, Тип_заготовки.вал);
Станок.create(1, "16К20", Тип_станка.токарный);

    Встроенные функции

    Язык использует встроенные математические функции языка Java, определенные в пакете Math.

    boolean приемлемость_погрешности(double значение_1, double значение_2) {
    return Math.abs(значение_1 - значение_2) <= 0.001;
}
